# 계좌시스템
## 프로젝트 설명

계좌 시스템은 사용자, 계좌, 거래 정보를 다룬다.


## 개발동기
- Spring Framework 에 익숙해지기
- API 설계 및 구현 연습
- 계획 중인 달러 기반 환투자 시스템의 모의연습
- API 별 테스트 코드 작성
